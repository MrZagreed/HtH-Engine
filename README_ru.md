# Spotify → Discord Lyrics Presence (модульная сборка)

**Одна кнопка**: запустите `start.cmd` (Windows) или `python run.py` (другие ОС).
При первом запуске проект:
1) создаст виртуальное окружение `duality_venv`,
2) поставит зависимости из `requirements.txt`,
3) перезапустит себя в venv,
4) откроет локальный HTTPS callback для Spotify и предложит авторизацию.

Особенности:
- Стабильный вывод: обновление не чаще 1 раза в секунду, 2–3 строки с вертикальным "пейджингом".
- Диагностика окружения и цветные логи (без эмодзи).
- Защита от сетевых сбоев, отсутствия Discord и рекламы в Spotify.
- Встроенный "warp" для сглаживания рассинхронизации с прогрессом Spotify.
